# Measurement System Analysis (MSA) & Gage R&R Case Study

## Overview

This case study demonstrates the application of Measurement System Analysis (MSA) in a manufacturing quality scenario.

The purpose of MSA is to verify that measurement systems are reliable before using inspection data for quality decisions.

A measurement system includes:

* Measuring equipment
* Operator
* Measurement method
* Environment
* Part being measured

---

# Problem Scenario

Product:
Bicycle Pedal

Inspection Requirement:

Hole Diameter:

20.00 ± 0.05 mm

Three operators measured the same component:

| Operator | Result   |
| -------- | -------- |
| A        | 20.01 mm |
| B        | 20.06 mm |
| C        | 20.03 mm |

The measurement variation indicates a possible measurement system issue.

---

# Possible Sources of Variation

## Repeatability

Variation caused by the measuring equipment.

Possible causes:

* Damaged measuring tool
* Poor calibration
* Equipment wear

## Reproducibility

Variation caused by operators.

Possible causes:

* Different measurement techniques
* Lack of standard work instruction
* Insufficient operator training

---

# Gage R&R Improvement Plan

If measurement variation is unacceptable:

1. Perform Gage R&R study
2. Analyze equipment and operator variation
3. Verify calibration status
4. Improve measurement method
5. Train operators
6. Repeat measurement validation
